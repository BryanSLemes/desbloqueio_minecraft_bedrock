Requisitos:

1. Ter o Minecraft instalado e já ter o aberto pelo menos uma vez.


Passo a Passo para desbloquear o Minecraft Bedrock:

1 - Execute o arquivo desbloqueio.bat como administrador, com o minecraft fechado;

2 - Ao finalizar a execução o programa vai abrir um prompt com algumas mensagens;

3 - Caso a mensagem final seja "Processo Concluido" seu Minecraft foi desbloqueado;

Informações sobre o programa:

Para efetuar o desbloqueio do Minecraft o programa:
    * Permite que a que administradores acessem a pasta C:\Program Files\WindowsApps;
    * Copiar o arquivo .\arquivo_desbloqueio\vcruntime140_1.dll para a pasta C:\Program Files\WindowsApps\MICROSOFT.MINECRAFTUWP...;
    * Copiar o arquivo .\arquivo_desbloqueio\MinecraftForFree.dll para a pasta C:\Users\conta\AppData\Roaming\Minecraft Bedrock\mods;

Toda a lógica do software está no arquivo desbloqueio.bat.
